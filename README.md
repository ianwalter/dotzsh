# dotzsh
> Ian's zsh configuration (part of [dotfiles][dotfilesUrl])

## Installation

Copies the zsh configuration to your home directory:

```console
./install.sh
```

## Update

Copies the zsh configuration back to the repo:

```console
./update.sh
```
